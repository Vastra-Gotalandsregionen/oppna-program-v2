# Governance

## Vad är Governance?

Kort sagt ett regelverk för hur beslut skall fattas.

* Vilka tekniker och ramverk som är tillåtna att använda
* Hur man begär förändringar på tillåtna tekniker, ramverk och versioner på dessa
